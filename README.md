# OKNC V3 Token

OKNC 权益凭证 V3 — BSC BEP20 代币

## 合约地址

`0x0da57c05760fa5F7EBcDdA89B01381a45EF75f1e`

## 核心参数

| 参数 | 值 |
|------|------|
| 名称 | OKNC Token |
| 符号 | OKNC |
| 小数位 | 18 |
| 总供应 | 21,000,000,000 (210亿) |
| 销毁 | 18,900,000,000 (189亿) |
| 置换池 | 2,100,000,000 (21亿) |

## 交易税

- 买入 1% → BNB 分红（RFI反射模型）
- 卖出 1% → 生态基金钱包
- 转账 0.5% → 生态基金钱包
- 添加/移除流动性 0%

## 总量机制

总量 210 亿在构造函数一次性铸造到合约地址，其中 189 亿立即销毁，剩余 21 亿作为旧代币置换池。

swapFromOld 从合约余额转账，不增发。总量永久锁定。
